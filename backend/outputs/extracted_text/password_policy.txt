PASSWORD SECURITY POLICY
Version: 1.0
Last Reviewed: January 2025
Purpose
This policy defines password management requirements for all employees.
Policy Statements
1. Employees must rotate passwords every 90 days.
2. Passwords must contain at least 12 characters.
3. Passwords must include uppercase letters, lowercase letters, numbers, and special characters.
4. Multi-factor authentication (MFA) is mandatory for all user accounts.
5. Passwords must never be shared with anyone.
6. Service account passwords must be stored securely using an approved password vault.
